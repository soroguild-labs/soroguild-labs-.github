# SoroGuild Labs

SoroGuild Labs builds open-source developer infrastructure for Stellar and Soroban applications.

Our goal is to help developers build, test, monitor, and launch Soroban dApps faster with reusable tools, starter kits, wallet components, RPC monitoring, and event indexing infrastructure.

## Projects

### SoroKit

A starter kit for building Soroban dApps with Next.js, TypeScript, wallet integration, Soroban RPC helpers, mock mode, tests, and documentation.

### Soro Wallet Kit

Reusable wallet connection hooks, adapters, and UI components for Stellar and Soroban applications.

### Soro RPC Monitor

A dashboard for monitoring Soroban RPC endpoint health, latency, network status, and availability.

### Soro Events Indexer

A starter backend service for fetching, parsing, storing, and exposing Soroban smart contract events through a simple API.

## Focus Areas

- Soroban developer tooling
- Wallet integration
- RPC reliability
- Smart contract event indexing
- Frontend starter kits
- Contributor-friendly open-source infrastructure
